# ТЗ с сайта
> У нас будут десятки волонтеров в разных городах. Координатору нужен 'штаб', чтобы видеть картину в целом. Функционал: 1) Учет волонтеров: Список команды, контакты, статус (активен/неактивен) 2) Статистика выполнения: Сколько заданий (лекций, мастер-классов) провел каждый волонтер 3) Календарь встреч: Сколько встреч/событий запланировано и проведено, явка 4) Рейтинг и достижения: Кто лидер по активности? У кого особые успехи? Визуализация в виде графиков или 'пьедестала почёта' 5) Удобство: Координатор должен заходить и видеть всё за 1 минуту

## Разбор
Если выражаться более простыми словами, от нас требуют сделать:
- сервис с двумя типами пользователей
- Базу данных с несколькими таблицами
- красиывый интрефейс к для управления
